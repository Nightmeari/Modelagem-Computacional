Repositório com códigos, exercícios e estudos desenvolvidos na disciplina de Modelagem Computacional.

## 📚 Conteúdo

Este repositório inclui:

- Implementações em Python de métodos numéricos
- Exercícios resolvidos com uso de séries de Taylor
- Visualizações gráficas utilizando matplotlib
- Experimentos computacionais relacionados à disciplina

## 🧠 Tópicos abordados

- Séries de Taylor (ordem 0 até n)
- Aproximações numéricas de funções
- Cálculo de erro relativo e erro percentual
- Análise de convergência
- Visualização de funções e aproximações

## 🛠️ Tecnologias utilizadas

- Python
- NumPy
- Matplotlib
- Jupyter Notebook / VS Code

## 📈 Objetivo

Este repositório tem como objetivo consolidar o aprendizado prático em modelagem computacional, explorando técnicas numéricas e análise de erros por meio de programação.
